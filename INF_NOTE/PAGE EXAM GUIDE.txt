# ⭐ 3️⃣ HOW TO MOTIVATE ANSWERS (EXAM GOLD ⭐)
Markers want:
👉 Plugin evidence
👉 Output interpretation
👉 Security meaning

## ✅ Example 1 — Suspicious process
❌ BAD
> process suspicious
✅ GOOD
> Using PsList, process evil.exe was observed. PsTree shows explorer.exe spawned it, which is abnormal. This suggests possible malware execution.

## ✅ Example 2 — Network connection
❌ BAD
> connected to IP
✅ GOOD
> NetScan shows PID 1234 connected to 185.x.x.x, an external address. This indicates possible command-and-control communication.

## ✅ Example 3 — Hidden process
✅ PERFECT
> PsScan identified PID 5555 not visible in PsList. This indicates DKOM hiding technique.
