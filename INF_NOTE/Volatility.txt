# ⭐ 1. PASSWORD INVESTIGATION (Credential Forensics)
## 🎯 Goal
Recover credentials, authentication artefacts, or evidence of password usage.
## ✅ Step 1 — Locate registry hives
bash
python3 vol.py -f memory.mem windows.registry.hivelist.HiveList

### Why
 Password hashes are stored in SAM
 Encryption key is in SYSTEM

👉 This is exactly what you did earlier.

## ✅ Step 2 — Dump registry hives
bash
python3 vol.py -f memory.mem windows.registry.hivelist.HiveList --dump

### Why
 Extract SAM + SYSTEM to disk so external tools can use them

## ✅ Step 3 — Extract password hashes
bash
impacket-secretsdump -sam SAM.hive -system SYSTEM.hive LOCAL

### Why
 Converts registry data into NTLM hashes

## ✅ Step 4 — Crack hashes (optional)
bash
hashcat -m 1000 hash.txt rockyou.txt

### What you achieve

✔ User accounts
✔ Password hashes
✔ Possible cracked passwords

## ⭐ 2. MALWARE INVESTIGATION
## 🎯 Goal
Find malicious processes, injected code, persistence, payloads.
## ✅ Step 1 — List processes
bash
python3 vol.py -f memory.mem windows.pslist.PsList

👉 Look for:
 Unknown names
 Wrong parent
 Suspicious start time

## ✅ Step 2 — Process tree
bash
python3 vol.py -f memory.mem windows.pstree.PsTree

👉 Detect:
 Malware spawning from explorer
 Office spawning cmd/powershell
 Orphan processes

## ✅ Step 3 — Malware injection detection
bash
python3 vol.py -f memory.mem windows.malfind.Malfind

### Why
 Detect injected executable memory
✔ Strong malware evidence

## ✅ Step 4 — Dump suspicious process
bash
python3 vol.py -f memory.mem windows.dumpfiles.DumpFiles --pid <PID>

### Why
 Extract malware binary
---

## ⭐ 3. VIRUS INVESTIGATION
