Step 1 — Memory Info for mem file

Volatility 3 does not have a plugin called windows.registry.hashdump.Hashdump. That was a Volatility 2 plugin.
In Volatility 3, the equivalent functionality is split into different plugins under windows.registry.*. For extracting user hashes (SAM + SYSTEM), you should use:
windows.registry.hivelist.HiveList
windows.registry.getcellroutine.GetCellRoutine
windows.registry.printkey.PrintKey

Step 1 — Find registry hives
python3 vol.py -f /home/kaliwg/Downloads/AdamFTriage.mem windows.registry.hivelist.HiveList
python3 vol.py -f /home/kaliwg/Downloads/AdamFTriage.mem windows.registry.hivelist.HiveList --dump     "This auto‑dumps all hives"

SYSTEM - \SystemRoot\System32\Config\SYSTEM
